Common Questions

The CSV isn't importing correctly.

Ensure your CSV uses UTF-8 encoding. The first column should be your "keys" (e.g., login_button_title).

Where do I find the generated files?

Translix saves the .strings files in the output directory you selected during the export process, organized by language code (en.lproj, etc.).
